 Privacy Policy for Dinamo Notebook

Last updated: 2026-01-20

Dinamo Notebook is a browser extension for football video analysis and tactical drawing.
Protecting user privacy is a core principle of this extension.

---

 Data Collection and Use

Dinamo Notebook does not collect, store, transmit, or sell personal data.

Specifically:
- No names, email addresses, or identifiers are collected
- No browsing history is collected
- No analytics or telemetry are used
- No tracking technologies are implemented
- No backend servers are operated

All functionality runs locally in the user's browser.

---

 User Content

All user-generated content (including drawings, annotations, projects, and analysis data) is:
- stored locally using browser storage
- never transmitted to the developer or third parties
- fully under the user's control

This data remains on the user's device unless the user explicitly exports it.

---

 AI Features and Third-Party Services

Dinamo Notebook offers optional AI-assisted analysis features.

- These features require the user to provide their own OpenAI API key
- API requests are sent directly from the user's browser to OpenAI
- Dinamo Notebook does not proxy, log, inspect, or store these requests
- Dinamo Notebook does not receive or store AI-generated responses

Use of AI features is subject to OpenAI's own Privacy Policy and Terms of Service.

---

 Video Platforms

Dinamo Notebook interacts only with video platforms already accessed by the user
(e.g. YouTube, Dailymotion, Twitch).

No additional data is collected beyond what is required to enable analysis tools on supported pages.

---

 Permissions Justification

Dinamo Notebook requests the following Chrome permissions:

- Storage
  Used to save projects and settings locally on the user's device.

- Active Tab
  Used to interact with the currently open video page.

- Tabs
  Used to detect supported video platforms.

These permissions are used solely to provide core functionality and not for data collection or tracking.

---

 Data Retention and Deletion

- All data remains on the user's device
- Users can delete their data at any time by:
  - deleting projects inside the extension
  - clearing browser storage
  - uninstalling the extension

Uninstalling the extension removes all locally stored data.

---

 Children's Privacy

Dinamo Notebook does not collect personal data from any users, including children.

---

 Changes to This Policy

Any changes to this Privacy Policy will be published with new versions of the extension.
There are no silent changes.
